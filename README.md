# 🧮 Tkinter Calculator App

A simple GUI-based calculator built using **Python** and **Tkinter**.  
This application performs basic arithmetic operations with a clean interface.

---

## 📌 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Percentage (%)
- Parentheses support `()`
- Clear (C) button
- Error handling for invalid expressions

---

## 🛠 Technologies Used

- Python 3.x
- Tkinter (Built-in GUI Library)

---

## 🧠 How It Works

- `StringVar()` updates the display dynamically.
- `lambda` functions pass button values.
- `eval()` evaluates the mathematical expression.
- `try-except` handles errors safely.

---

## ⚠️ Important Note

This calculator uses Python's `eval()` function.  
Avoid entering unsafe or non-mathematical expressions.
